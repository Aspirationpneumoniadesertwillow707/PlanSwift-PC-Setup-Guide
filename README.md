## 🛠️ Resolving Setup Failures

Installation issues often stem from missing components or permission blocks. Try these steps if the setup fails.

### Run in Compatibility Mode
If the installer closes unexpectedly, use compatibility settings. Right-click the installer file, select Properties, and open the Compatibility tab. Check the box for Run this program in compatibility mode for and choose Windows 10 from the list. Click Apply and run the file again.

### Update the .NET Framework
PlanSwift requires specific versions of the Microsoft .NET Framework. If the installer hangs, visit the official Microsoft website to download the latest .NET Desktop Runtime. Restart your computer after the install, then try the PlanSwift setup again.

### Manage Antivirus Interference
Sometimes, security software blocks the PlanSwift installer. Temporarily disable your real-time virus protection during the installation. Re-enable the protection once the setup finishes.

### Clear Temporary Files
Corrupt temporary files can disrupt a new install. Press the Windows key + R, type %temp%, and press Enter. Delete all items in the folder that opens. Restart the computer and run the PlanSwift installer once more.

## 📧 Common Error Codes

**Error: Access Denied**
This error occurs when the user lacks administrative rights. Ensure you are signed in to a Windows administrator account. Right-click the setup file and select Run as administrator every time you attempt the install.

**Error: MsiExec Failure**
This points to issues with the Windows Installer service. Open the Task Manager, click on the Services tab, and locate msiserver. Right-click it and select Restart.

**Error: Missing DLL Files**
If the application fails to open after install, it means a system file failed to register. Perform a clean uninstall, restart the system, and install the software again using the Run as administrator option.
